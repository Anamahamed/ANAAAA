# نظام الإدارة والإصدار الذكي (SID System)

## نظرة عامة

نظام محلي متكامل يهدف إلى إدارة بيانات وهويات الأفراد، وإصدار بطاقات تعريفية احترافية بدقة عالية مع ضمان الأمان والبحث والتوثيق السريع، يعمل بشكل كامل دون الحاجة للإنترنت.

## المكونات الأساسية
- **قاعدة بيانات Excel** لتنظيم المعلومات، منع التكرار، وحساب حالات الوثائق تلقائياً.
- **قالب Word أوتوماتيكي** لإصدار بطاقات تعريفية جاهزة للطباعة مع الصورة والباركود.
- **نظام أرشفة وتوثيق**: توليد QR Code لكل بطاقة وسجل بحث فوري للوصول للبيانات.

## المميزات
- التحقق الشرطي لمنع التكرار والأخطاء.
- إصدار بطاقات فورية بدمج بيانات Excel مع قوالب Word والصور الشخصية (دمج مراسلات).
- يعمل 100% دون إنترنت لضمان الأمان وسرية المعلومات.
- سهولة التعديل والتحديث الجماعي للبيانات.
- أرشفة رقمية متقدمة وإمكانية البحث السريع عبر QR أو رقم البطاقة.

## الاستخدام
1. إدخال بيانات الأفراد في قاعدة بيانات Excel حسب الحقول المحددة.
2. استيراد البيانات تلقائياً في قالب Word لإصدار البطاقة وطباعتها.
3. توليد باركود لكل بطاقة لأغراض التوثيق والبحث.
4. تحديث الحالات تلقائياً وحماية البيانات من التكرار.

## الحقول الأساسية للبطاقة
- رقم البطاقة (فريد)
- تاريخ الإصدار / تاريخ الانتهاء
- الباركود / QR
- الاسم رباعي عربي/إنجليزي
- تاريخ ومكان الميلاد
- الجنسية والعرق
- المهنة
- مكان العمل
- الصورة الشخصية
- رقم قانوني متكرر على الجهة الخلفية
- تعليمات الاستخدام

## خطة العمل القادمة
- رفع نموذج قاعدة البيانات على Excel.
- إضافة قوالب Word للبطاقات التعريفية.
- إرفاق نصوص VBA لأتمتة العمليات (تحديث تلقائي – حماية – توليد QR...).
- شرح استخدام النظام خطوة بخطوة.
